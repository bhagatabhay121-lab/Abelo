## [1.0.0] — 2026-06-28

### 🎉 Initial Release

#### Added
- **Home Screen** — Personalised feed with Most Played, Top Charts, New Releases, Editorial Picks, and Trending sections
- **JioSaavn Streaming** — Direct on-device API client with DES/ECB stream URL decryption (no server required)
- **Search** — Real-time search across songs, albums, and artists with Popular Artists, Albums, and Trending sections
- **Library** — Now Playing queue, Last Session, Playlists, and Device Music hub
- **Local Music** — Scan device for audio files; browse by Songs, Albums, Artists, Playlists
- **File Import** — Import audio files directly via system file picker
- **Downloads** — Download any song for offline playback with per-song progress tracking
- **YouTube Playback** — Search and stream songs from YouTube
- **Background Playback** — Continues playing when app is in background or screen is off
- **Lock-Screen Player** — Full-screen player overlay on the lock screen (Android 14+ `USE_FULL_SCREEN_INTENT` compatible)
- **Media Notification** — Rich notification with artwork, controls, and seek bar
- **Dark / Light / System Theme** — Full theming support
- **Accent Colour & Hue** — Customisable accent colour
- **Glass-morphism UI** — Frosted-glass widgets throughout
- **Shimmer Loading** — Skeleton placeholders while content loads
- **Mini Player** — Persistent bottom player with like, play/pause, next
- **Onboarding** — First-launch personalisation screen
- **Settings** — Theme, UI, Playback, Language, Backup & Restore, About
- **Playlist Management** — Create, edit, delete, and share playlists
- **Backup & Restore** — Export and import all playlists and settings
- **Home-Screen Widget** — Playback controls from the Android home screen
